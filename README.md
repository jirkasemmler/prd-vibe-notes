# Vibe Coding Workshop — Instructor Materials

Materiály pro lektora workshopu "Od nápadu k deploynuté appce za 3 hodiny"
na konferenci Hack Your Way 2026 (Brno, 28. dubna).

## Obsah

| Soubor | Popis |
|--------|-------|
| `workshop-guide.md` | Kompletní kostra workshopu — co říct, co ukázat, timing |
| `workshop-slides.pptx` | Prezentace (10 slidů) |
| `cheatsheet/` | Cheat sheet pro účastníky (k vytištění) |
| `demo/` | Připravená demo appka pro úvodní ukázku |

## Účastnický kit

Účastníci klonují separátní repo: `workshop-kit`
(obsahuje CLAUDE.md, custom commands pro Claude Code, README s quick startem)

## Checklist před workshopem

- [ ] Pushni `workshop-kit` repo na GitHub, doplň URL do guide
- [ ] Připrav demo appku (deploynutou na Vercelu) do `demo/`
- [ ] Otestuj celý flow end-to-end na čistém stroji
- [ ] Vytiskni cheat sheety
- [ ] Připrav fallback Supabase projekt
- [ ] Pošli email s prerekvizitami (7 dní předem)
- [ ] Pošli reminder email (2 dny předem)
- [ ] Ověř Wi-Fi na místě + mít mobilní hotspot jako backup
